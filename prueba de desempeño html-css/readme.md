This project aims to assess the skills and knowledge acquired during module 2, corresponding to HTMKL and CSS

 Technologies Used
    • HTML5 – Semantic structure 
    • CSS3 – Styling, layout, and responsive design 
    • Google Fonts – Overpass and Ubuntu 

 Project Structure
blogr-landing-page-main/
│
├── index.html
├── styles.css
└── assets/
  

##  Main Features

- Header 
- Navigation menu with dropdowns built using only CSS
- Reusable buttons with hover states
- Layout distribution using Flexbox
- Typography and color palette faithful to the original design

---


##  Design

The design is based on the official Frontend Mentor challengesigma challenge
- Colors defined using the HSL color system
- Characteristic curved borders on the header and sections
- SVG background patterns positioned using multiple background layers

---

##  Installation and Usage

1. Clone or download this repository https://github.com/Julian-1412/html-css
2. Open the folder "prueba de desempeño html-css"
3. Open`index.html` file in your browser
4. No additional dependencies or configuration required

---

##  Learnings

During the development of this project, the following concepts were reinforced:

- Advanced use of `background-image` with multiple layers
- Proper handling of relative paths in CSS
- CSS code organization and cleanliness
